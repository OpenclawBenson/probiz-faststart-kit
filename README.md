# ProBiz FastStart Kit

Complete recruiting system for new ProBiz members.

## Product Details

- **Price:** $37
- **Format:** Digital download (PDFs, templates, video)
- **Delivery:** Instant download after purchase

## What's Included

1. Email Swipe File (10 recruitment emails)
2. First Conversation Script
3. Social Media Post Templates (20 posts)
4. Contractor Tracker Spreadsheet
5. 30-Day FastStart Roadmap
6. Quick-Start Walkthrough Video

## Tech Stack

- Next.js
- Vercel
- Stripe (for payments)

## Development

```bash
npm install
npm run dev
```

## Deployment

Automatically deploys to Vercel on push to main branch.
